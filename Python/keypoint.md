## escape character 이스케이프 문자 : (\)

### 문자열 내부에 따옴표 넣기

```
>>> print('"안녕하세요"라고 말했습니다.')
>>> print("\"안녕하세요\"라고 말했습니다.") #이스케이프 문자 사용
"안녕하세요"라고 말했습니다.
```

### 탭 사용 : (\t)

### 줄바꿈 사용 : (\n)

### 역슬래시 (\) 출력

```
>>> print("\\ \\ \\ \\")
\ \ \ \
```
